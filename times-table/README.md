# times-table

return a time-table in clojure.

## LazySeq
this code gives a lazy-sequence.
```clojure
(map #(* n %1) (range 11))
```
So - when is concatenated with a string does not work as expected.
so we can force evaluation using into. 
```clojure
(into '() (map #(* n %1) (range 11)))
```
